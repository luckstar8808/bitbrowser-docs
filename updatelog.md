2. 代理检测 IP 信息调整，增加显示时区、经纬度等更多 IP 信息
3. 新增偏好设置，可设置浏览器是否禁用图片以及网络不通是否继续打开，可在系统设置中设置偏好设置，后续创建的窗口，都将优先应用全局偏好
4. 优化指纹项目
5. Local Api /browser/open 接口新增 args 参数，可自定义传入浏览器启动参数

### 比特浏览器 2.2.5 升级日志（2021-10-20）

1. 修复浏览器书签栏，扩展无法固定问题

3. 新增仿真输入功能，可到系统设置与扩展中心，启用仿真输入插件
4. 新增模拟 IOS 指纹，优化 Android 指纹配置

7. 创建修改窗口，账号平台选择【其他】时，允许留空，留空则只打开工作台页面
